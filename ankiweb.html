That's a great idea in general and especially if you want to try out a newly released add-on that changes due dates and that doesn't have many upvotes ... Use it at your own risk. This add-on adds a context menu to the Add button which is in the lower left corner of the Add window. You can change the due number for the cards of the note you just created or you can bury those cards. This might be useful if you add several cards that ask about the same topic in different ways and you don't want to see them at the same time. You could quickly reschedule them when you see them the first time in the reviewer (with ReMemorize/QuickReschedule or MoreButtonsForNewCards) but when I only reschedule later during a review I will have forgotten which cards need to be rescheduled ... You can adjust some settings in the add-on dialog and also have the option to reschedule cards shown. This is not intended for regular use since this graduates cards and pressing again "Again" on such review cards reduces their ease which might put you into "ease hell" in the long run. Maybe relevant if you know the card is very easy - but in this case you could just use the add-on More Answer Buttons for New Cards (a version for 2.1 is available here) or with lovac42's ReMemorizeButtons. other add-ons If you want all new cards to be shown only after n days use lovac42's 3ft Under: Bury Newly Added Cards.
